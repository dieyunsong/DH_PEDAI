# DH_PEDAI

Materials for the **Digital Humanities Summer Institute (DHSI) 2026** course on pedagogy and AI in the digital humanities.

## Repository Contents

```
├── Porfolio/          # Instructor CV and background
├── readings/
│   ├── commandlines_humanities.pdf   # Course reading on command-line methods
│   ├── sf_quiz.html                  # Science fiction quiz exercise
│   └── Distant_reading/             # Texts and resources for distant reading lab
│       ├── 100_diverse_science_fiction_novels.md
│       └── pg*.txt                  # Project Gutenberg science fiction texts
```

## About

This repository supports a DHSI 2026 course exploring digital humanities methods — including distant reading and text analysis — with a focus on pedagogy and AI-assisted approaches.
